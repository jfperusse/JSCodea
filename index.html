Lost Controls - User Manual

Goal

Get the robot (@) to the end of the level (*).

not so lost Controls

Enter, Space: Skip text
Arrow Keys, WASD: Move around
1-3: Drop module
4-6: Take module

Legend

@: Robot (Player)
*: Goal
#: Wall
NSWE: Direction modules (North, South, West, East)
